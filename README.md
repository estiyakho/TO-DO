# 📋 Viewdue

`Viewdue` is a **local-first task manager** that helps you organize todos, review progress, and track completion trends.
Designed for mobile devices, it has a clean, tabbed interface for easy task management.

## ✨ What The App Does

`Viewdue` has **five main sections**:

* **Categories** – create and browse task categories and track progress.
* **All Todos** – add tasks, filter by status or category, search, and sort.
* **Calendar** – view tasks by day in a monthly calendar.
* **Statistics** – see totals, completion rates, weekly activity, and streaks.
* **Settings** – customize themes, colors, time format, default screen, and reset data or stats.

All data is stored locally on your device. **No internet connection or backend is required.**

---

## 🔹 Features

* Add and delete tasks
* Mark tasks as `Doing` or `Done`
* Create custom categories with colors and icons
* Search tasks and categories
* Sort tasks by newest, oldest, or title
* Track progress per category and overall
* Calendar view for easy task planning
* Track daily, weekly, and total statistics
* Customize theme, accent colors, and language
* Reset app data or statistics when needed

---

## 📲 Platform & Requirements

* Works on Android and iOS devices
* **Minimum Android version:** Android 8.0 (Oreo)
* iOS 15.1 or higher
* Portrait orientation only

---

## 🚀 Getting Started

1. Install dependencies:

```bash
npm install
```

2. Start the app:

```bash
npm start
```

3. Run on your device:

```bash
npm run android
npm run ios
npm run web
```

---

## 🗂 Project Structure

* `app/` – screens and navigation
* `components/` – reusable UI components
* `store/` – local data storage
* `types/` – data types
* `utils/` – helpers and shared logic
* `assets/` – icons, images, and splash screens
